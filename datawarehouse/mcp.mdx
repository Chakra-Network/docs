## Intro

Chakra supports Anthropic's [Model Context Protocol](https://modelcontextprotocol.io/introduction) (MCP), allowing users to integrate data natively into existing chat clients.

More details on this release can be found [here](https://www.chakra.dev/writing/chakra-now-supports-model-context-protocol-mcp).

With MCP, existing chat experiences can be augmented with data from the marketplace along with an organization's proprietary data.
